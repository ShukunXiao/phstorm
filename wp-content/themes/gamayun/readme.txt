== Description ==
Gamayun is simple WordPress theme with grid-layout, extra menu, 3 post sliders and google typography, easy to customize with Customizer theme options. Donate and support theme: https://paypal.me/zoranstosic or, send your donation directly to my Paypal account: zstosic@gmail.com. If you need more information please contact me at Viber/Telegram: +381 637618511.


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Now go to "Appearance" -> "Customize" to configure theme options.
